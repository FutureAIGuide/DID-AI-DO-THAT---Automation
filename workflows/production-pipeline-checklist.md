# Production Pipeline Checklist

1. Load approved source exports.
2. Run Topic Hunter.
3. Score candidate topics.
4. Select the top story.
5. Build the research packet.
6. Generate the article outline.
7. Generate the article draft.
8. Run the draft verification gate for claims, citations, and legal risk.
9. Resolve all draft HOLD or REJECT findings.
10. Approve and save the final article.
11. Generate the newsletter draft.
12. Generate the social package.
13. Generate the video package.
14. Generate the thumbnail and social-image briefs.
15. Generate the prompt pad.
16. Run the package verification gate across all derivative assets.
17. Resolve all package HOLD or REJECT findings.
18. Build the publishing package and promote approved drafts to final.
19. Run the publish-pipeline validation.
20. Publish or schedule approved assets.
21. Archive completed assets while retaining source material.

No validation or verification step may be skipped.
